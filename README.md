# cf-visibility-check

![build status](https://goo.gl/1l8CNJ)

Check whether an entity is considered visible.

## Installation

      npm install cf-visibility-check

## Usage

      var isVisible = require('cf-visibility-check')

      if (isVisible(entity)) {
        console.log('Entity is visible!')
      }
